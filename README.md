# 🏥 BooknMeet - Hospital & Appointment Finder

BooknMeet is a modern web application that helps users **find hospitals, explore doctors, and book appointments بسهولة (easily)**. It provides smart filtering, real-time search, and a smooth booking experience.

---

## 🚀 Features

✨ **Smart Search System**

* Search hospitals by location, specialty, rating, budget, and availability
* Dynamic filtering with real-time results

👨‍⚕️ **Doctor & Hospital Listings**

* View hospital details, ratings, and reviews
* See available doctors by specialty

📅 **Appointment Booking**

* Select doctor, date & time
* Easy booking with instant confirmation

🔐 **Authentication System**

* Login & Register (Patient / Doctor roles)
* Local storage-based user system

📊 **Advanced Filters**

* ⭐ Rating filter
* 💰 Budget slider
* 🕐 Availability (Morning, Evening, 24h)

📱 **Responsive Design**

* Works on desktop, tablet, and mobile

---

## 🛠️ Tech Stack

## ⚙️ How It Works

1. User opens the website
2. Searches hospitals using filters
3. Views hospital details
4. Clicks **Book Appointment**
5. Selects doctor & time
6. Appointment gets confirmed
